# 读心赘婿：王府风云 - 游戏设计文档

## 1. 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 读心赘婿：王府风云 |
| 游戏类型 | 古装穿越 + 权谋策略 + 视觉小说 + 后宫养成 |
| 核心玩法 | 读心系统 + 宫廷权谋 + 商业经营 + 恋爱养成 |
| 目标玩家 | 16-35岁男性玩家 |
| 年龄分级 | R15（涉及权谋与情感） |
| 游戏平台 | PC / 移动端 |
| 预计时长 | 30-50小时 |

### 核心理念
> "现代社畜穿越成大宁国永王府的废物赘婿，意外获得读心术能力。他能听到所有人的内心真实想法——包括那三个对他冷漠的美娇妻。利用这个金手指，他一步步化解王府危机，在朝堂上如鱼得水，最终征服三颗芳心，成为人人羡慕的王府主宰。"

### 玩法特色
- **读心系统**：可偷听所有人的心声，获取情报和秘密
- **权谋策略**：在朝堂和商战中运用情报优势
- **三女主线**：三位性格迥异的王妃，每条独特剧情
- **王府经营**：管理王府产业，提升声望和财富
- **多重结局**：不同选择导向不同结局

## 2. 世界观设定

### 2.1 时代背景

**【大宁国】**

架空古代王朝，约相当于唐宋时期的繁华盛世。

**政治背景**：
- 皇帝年迈，皇子夺嫡之争暗流涌动
- 外有北蛮、西夏等强敌环伺
- 内有权臣当道，党争激烈
- 商贸繁荣，但官商勾结严重

**地理设定**：

```
大宁国疆域：
┌─────────────────────────────────────┐
│         北境（北蛮）                  │
│   游牧民族，年年南侵                  │
├─────────────────────────────────────┤
│         京城（天京）                  │
│   皇宫、各王府、权贵府邸              │
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐  │
│  │     永王府（主角所在）          │  │
│  │  位于京城繁华地段，占地百亩     │  │
│  └───────────────────────────────┘  │
├─────────────────────────────────────┤
│         江南（经济重镇）              │
│   商贸发达，丝绸、茶叶产地           │
├─────────────────────────────────────┤
│         沿海地区（对外贸易）          │
│   海上丝绸之路起点                   │
└─────────────────────────────────────┘
```

### 2.2 永王府设定

**【王府地位】**

| 属性 | 描述 |
|------|------|
| 王爵 | 水亲王（永王）萧景 |
| 地位 | 皇三子，母妃出身不高，不受重视 |
| 兵权 | 无实权兵权，只有王府亲兵300 |
| 财力 | 中等，有家族产业但经营不善 |
| 声望 | 一般，被认为是"闲散王爷" |

**【王府危机】**

- **外部威胁**：太子党视永王为眼中钉
- **内部问题**：产业亏损严重，入不敷出
- **家庭矛盾**：三妃不和，内斗消耗
- **政治风险**：随时可能被卷入夺嫡之争

### 2.3 读心术设定

**【系统：读心神瞳】**

主角穿越后觉醒的能力：

| 功能 | 描述 | 限制 |
|------|------|------|
| 听心声 | 可听到半径50米内所有人的内心想法 | 距离限制 |
| 看情绪 | 可看到他人情绪颜色（红怒/绿喜/蓝悲） | 被动能力 |
| 探秘密 | 深度探测可发现隐藏的秘密 | 消耗精力 |
| 察谎言 | 自动识别谎言 | 被动能力 |
| 感暗示 | 感知他人对自己的真实态度 | 被动能力 |

**【读心限制】**：
- 精力值：每次主动读心消耗精力
- 距离：最多50米范围
- 遮蔽：某些修炼之人可屏蔽读心
- 过载：读心过多会头痛、眩晕
- 天命：无法读取皇帝和某些特殊人物

**【心声类型】**

| 类型 | 颜色 | 示例 |
|------|------|------|
| 真实想法 | 白色 | "这个人是谁？" |
| 阴谋诡计 | 紫色 | "等机会到了就..." |
| 爱慕之情 | 粉色 | "他今天真帅..." |
| 杀意 | 黑色 | "我要杀了他！" |
| 贪婪 | 金色 | "这个宝贝是我的..." |
| 恐惧 | 灰色 | "不要伤害我..." |

## 3. 角色设计

### 3.1 主角

**林云（Lin Yun）**

| 属性 | 详情 |
|------|------|
| 现代身份 | 28岁社畜，某上市公司高管助理 |
| 古代身份 | 永王府赘婿，出身寒门的书生 |
| 年龄 | 25岁 |
| 外貌 | 英俊儒雅，身材修长，气质温润 |

**性格特点**：
- **表面**：温和谦逊，谨小慎微，看似软弱
- **内在**：精明强干，城府深沉，有仇必报
- **优点**：善于察言观色，擅长谋划，重情义
- **缺点**：初来乍到，缺乏古代社会经验

**穿越原因**：
在现代加班猝死，灵魂穿越到古代刚入赘永王府的书生身上。

**原主处境**：
- 入赘永王府三年，被视为废物
- 三位王妃都对他冷漠
- 王府上下看不起他
- 随时可能被休妻赶出王府

**穿越后获得**：
- 读心术能力
- 现代人的知识和思维
- 前世职场的经验

**能力树**：

```
【读心系】
├─ 基础读心（被动）
├─ 深度探测（主动）
├─ 群体读心（高级）
└─ 心灵感应（顶级）

【权谋系】
├─ 察言观色
├─ 谋略规划
├─ 谈判说服
└─ 舆论操控

【商业系】
├─ 商业洞察
├─ 经营管理
├─ 谈判交易
└─ 创新思维

【魅力系】
├─ 谈吐风趣
├─ 才艺展示
├─ 武艺入门
└─ 气质提升
```
