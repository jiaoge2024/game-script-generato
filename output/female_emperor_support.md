# 大梁女帝：帝师之路 - 游戏设计文档

## 1. 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 大梁女帝：帝师之路 |
| 游戏类型 | 历史策略 + 视觉小说 + 宫廷权谋 |
| 核心玩法 | 穿越辅佐 + 朝堂斗争 + 军事征战 + 宫廷恋爱 |
| 目标玩家 | 16-40岁玩家 |
| 年龄分级 | R15（涉及政治权谋与情感纠葛） |
| 游戏平台 | PC / 移动端 |
| 预计时长 | 50-80小时 |

### 核心理念
> "现代历史系研究生穿越到大梁王朝，成为女帝的帝师。内斗权臣、外御强敌、改革弊政、辅佐明君。在权力的漩涡中，与女帝产生超越君臣的情感羁绊。最终的选择是：功成身退，还是...权倾天下？"

### 玩法特色
- **历史穿越**：现代知识与古代智慧的碰撞
- **朝堂权谋**：与各派系斗智斗勇，平衡朝局
- **军事征战**：指挥军队，抵御外敌，开疆拓土
- **改革变法**：推行新政，改变国家命运
- **情感羁绊**：与女帝及多位女性角色发展情感
- **多重结局**：不同选择导向截然不同的结局

## 2. 世界观设定

### 2.1 时代背景

**【大梁王朝】**

架空历史王朝，原型参考唐宋时期，约相当于公元950-1050年的技术水平。

**政治背景**：
- 大梁立国150年，正值盛转衰的转折点
- 前代皇帝昏庸，留下烂摊子
- 新帝继位不久，根基未稳
- 外有强敌环伺，内有藩镇割据

**地理设定**：


**苏云（Su Yun）**

| 属性 | 详情 |
|------|------|
| 现代身份 | 历史系研究生，专攻中国古代史 |

| 政治智慧 | 权谋策略、治国理念 | 被动 |
| 谈判技巧 | 提升说服成功率 | 被动 |
| 医学常识 | 应对疫情、伤病 | 先知点 |

**古代能力**（穿越后获得）：
| 能力 | 效果 | 来源 |
|------|------|------|
| 诗词歌赋 | 获得文人认可 | 系统赠送 |
| 武艺骑射 | 自保能力 | 刻苦训练 |
| 书法绘画 | 结交权贵 | 系统赠送 |
| 兵法韬略 | 军事指挥 | 研读兵书 |

**【系统能力：帝师系统】**

主角穿越后激活的系统：

| 系统模块 | 功能 | 消耗 |
|---------|------|------|
| 历史预知 | 可预知未来重大事件 | 先知点 |
| 人才识别 | 察看他人忠诚度和能力 | 先知点 |
| 科技树 | 解锁现代科技 | 先知点 |
| 谋略推演 | 模拟政策效果 | 先知点 |
| 人格魅力 | 被动提升好感度 | 被动 |
| 危机预警 | 提前预知危险 | 免费触发 |

**先知点获取**：
- 完成主线剧情
- 推行成功改革
- 获得重要胜利
- 女帝好感度提升

### 3.2 女帝（主要女主角）

#### 萧长歌（Xiao Changge）- 大梁女帝

| 属性 | 详情 |
|------|------|
| 年龄 | 22岁（登基2年） |
| 外貌 | 凤目樱唇，容貌绝美，气质雍容华贵 |
| 身材 | 修长挺拔，170cm左右 |

**性格特点**：
- **表面**：威严冷峻，帝王威仪，难以接近
- **内心**：孤独敏感，渴望理解，有少女柔情
- **理想**：成为一代明君，中兴大梁
- **软肋**：对主角有超越君臣的情感，但克制

**背景故事**：
- 先帝唯一子嗣，在激烈的夺嫡之争中胜出
- 登基时年仅20岁，面临内忧外患
- 自幼接受帝王教育，性格坚韧
- 从未有人真正理解她内心的孤独

**历史原型**：融合武则天、宋仁宗、明成祖等形象

**与主角的关系发展**：
```
初识：君臣之别
    ↓
赏识：帝师与女帝
    ↓
依赖：精神支柱
    ↓
情愫：超越君臣
    ↓
抉择：江山与爱情
```

**攻略难度**：★★★★★
- 需要先建立政治信任
- 需要展现真才实学
- 需要理解她的孤独和压力
- 最终面临江山与爱情的两难选择

**特殊剧情**：
- 【御书房夜谈】：深夜讨论国事，情感升温
- 【微服私访】：一起出宫体察民情
- 【生死与共】：共同经历刺杀或战争
- 【帝后大婚】：可能成为大梁第一任男后

**【音效：时空穿越音效】**

**场景2：大梁天京城**

> 地点：天京城外破庙

**场景3：初入朝堂**

> 时间：永明三年春
> 地点：天京城贡院

**【事件：殿试】**

皇帝驾崩，女帝登基，举行首次科举考试。

**苏云**：（内心）这是机会！利用历史知识和现代理念，一定能脱颖而出！
