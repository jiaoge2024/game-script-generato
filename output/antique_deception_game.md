# 古董局中局 - 鉴宝风云录游戏设计文档

## 1. 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 古董局中局 - 鉴宝风云录 |
| 游戏类型 | 悬疑推理 + 模拟经营 + 视觉小说 + 收集养成 |
| 核心玩法 | 古董鉴定 + 造假技术 + 江湖争斗 + 悬疑破局 |
| 目标玩家 | 16-40岁玩家 |
| 年龄分级 | R15（涉及犯罪与复杂人性） |
| 游戏平台 | PC / 移动端 |
| 预计时长 | 40-60小时 |

### 核心理念
> "三十年前，你的师父——一代鉴宝宗师，被陷害鉴定赝品，身败名裂，含恨而终。三十年后，你带着师父传授的绝学归来，誓要揭开当年真相，让真正的幕后黑手付出代价。在古董江湖这个真假难辨的世界里，你将以假乱真，以真辨假，成为新一代鉴宝传奇。"

### 玩法特色
- **鉴定系统**：学习古董知识，辨别真伪
- **造假系统**：掌握造假技艺，以假乱真
- **局中局**：设局与破局，智力博弈
- **江湖门派**：五大流派，明争暗斗
- **商战系统**：拍卖会、古董店经营
- **悬疑主线**：抽丝剥茧，揭开真相

## 2. 世界观设定

### 2.1 古董江湖

**【江湖格局】**

古董界分为五大流派，各有专长：

| 流派 | 名称 | 擅长 | 代表人物 | 立场 |
|------|------|------|---------|------|
| 鉴宝派 | "眼"派 | 鉴定古董真伪 | 许愿（主角师父，已故） | 正统 |
| 造假派 | "手"派 | 制作高仿赝品 | 老朝奉（反派） | 邪派 |
| 修补派 | "工"派 | 修复残损古董 | 木工许 | 中立 |
| 收藏派 | "藏"派 | 收藏珍品古董 | 秦老 | 正派 |

## 4. 核心玩法

### 4.1 鉴定系统

**【鉴定流程】**

```
观察古董
    ↓
选择鉴定工具
    ↓
进行鉴定步骤
    ↓
得出鉴定结论
    ↓
判断真伪/年代/价值
```

**【鉴定工具】**

| 工具 | 作用 | 精度 |
|------|------|------|
| 肉眼 | 初步观察 | ★☆☆☆☆ |
| 放大镜 | 细节观察 | ★★☆☆☆ |
| 手电筒 | 透光检查 | ★★☆☆☆ |
| 紫光灯 | 荧光反应 | ★★★☆☆ |
| 放大镜 | 高倍观察 | ★★★★☆ |
| 专业设备 | 科学检测 | ★★★★★ |

**【鉴定要点】**

不同品类古董的鉴定要点：

| 品类 | 鉴定要点 | 常见造假 |
|------|---------|---------|
| 瓷器 | 釉面、胎质、款识、底足 | 做旧、接底、后挂彩 |
| 玉器 | 质地、雕工、包浆、沁色 | 仿沁、老玉新工 |
| 字画 | 纸张、墨色、印章、题跋 | 老纸新画、仿印 |
| 青铜器 | 锈蚀、铸造痕迹、包浆 | 假锈、仿铸 |
| 杂项 | 工艺、材质、使用痕迹 | 现代仿品 |

---

### 4.2 造假系统

**【造假技艺】**

主角可以学习造假技艺（用于设局）：

| 技能 | 描述 | 熟练度 |
|------|------|--------|
| 做旧 | 让新物品看起来有年代感 | 入门-大师 |
| 仿釉 | 仿制瓷器釉面 | 入门-大师 |
| 刻印 | 伪造印章款识 | 入门-大师 |
| 拼接 | 将真品和赝品拼接 | 入门-大师 |
| 移花接木 | 将真品部件移到赝品上 | 入门-大师 |

**【造假工坊】**

玩家可以开设工坊，生产赝品：
- 招募工匠
- 购买材料
- 研制配方
- 生产赝品

**【道德抉择】**
- 造假可以赚钱，但会降低道德值
- 道德值过低会影响结局
- 某些事件需要用赝品设局

---

### 4.3 局中局系统

**【设局】**

玩家可以设局对付敌人：

| 局的类型 | 描述 | 风险 | 收益 |
|---------|------|------|------|
| 鱼目混珠 | 用赝品冒充真品卖给敌人 | 高 | 财富+声望 |
| 引君入瓮 | 诱使敌人买入赝品 | 中 | 打击敌人 |
| 将计就计 | 利用敌人的局反制 | 极高 | 彻底击败 |
| 瞒天过海 | 偷梁换柱 | 高 | 获得珍品 |

**【破局】**

敌人也会设局，玩家需要破局：

1. **识别设局**：通过鉴定和情报发现异常
2. **分析意图**：理解对方的真正目的
3. **制定策略**：选择应对方式
4. **执行破局**：反制或逃脱

---

**拍卖流程**：

4. 转手获利

## 9. 对话脚本示例

### 9.1 鉴定场景

**场景**：古董店鉴定

**客人**：老板，您给掌掌眼，这个花瓶值多少钱？

**许朝**：（接过花瓶）

**【鉴定流程】**
1. 观察器型：梅瓶，撇口，束颈，丰肩
2. 观察釉面：青花发色浓艳，有铁锈斑
3. 观察胎质：白中泛青，细腻致密
4. 观察纹饰：缠枝莲纹，笔法流畅
5. 观察底足：露胎，有火石红

**许朝**：这是元代青花梅瓶...

**客人**：（激动）真的？！

**许朝**：但是...

**客人**：但是什么？

**许朝**：（指出破绽）底足的火石红是后天做上去的，而且青花的铁锈斑也太规矩了...

**客人**：（脸色发白）那...

**许朝**：是现代仿品，价值大概五百左右。

**客人**：（失望）谢了...
