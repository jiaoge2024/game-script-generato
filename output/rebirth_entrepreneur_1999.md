# 重生1999：商途红颜 - 游戏设计文档

## 1. 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 重生1999：商途红颜 |
| 游戏类型 | 视觉小说 + 模拟经营 + RPG |
| 核心玩法 | 重生逆袭 + 商业经营 + 角色攻略 |
| 目标玩家 | 18-35岁男性玩家 |
| 年龄分级 | R18（建议18岁以上） |
| 游戏平台 | PC / 移动端 |
| 预计时长 | 40-60小时 |

### 核心理念
> "前世窝囊一世，被妻子背叛害死。重回1999年，这一世，我要掌握自己的命运，建立商业帝国，让那些伤害我的人付出代价，也让属于我的红颜知己环绕身侧。"

### 玩法特色
- **时间线系统**：从1999年初中开始，经历高中、大学、创业期，到2030年建立商业帝国
- **商业经营**：从倒卖小商品开始，逐步涉足房地产、互联网、金融、科技等领域
- **系统技能**：主角拥有"未来先知系统"，可获取未来商业信息和特殊技能
- **后宫养成**：可攻略10+位不同类型的女性角色，包括前世背叛的妻子
- **剧情分支**：多种结局路线，商业帝国规模和后宫人数影响结局

## 2. 世界观设定

### 2.1 时代背景

**时间线：1999年-2030年**

游戏重现中国改革开放后的黄金时代，涵盖：
- 1999-2003：互联网泡沫前后、中国加入WTO
- 2004-2008：房地产腾飞、北京奥运
- 2009-2014：移动互联网兴起、微信时代
- 2015-2020：共享经济、AI浪潮
- 2021-2030：元宇宙、新能源、量子计算

**地理设定**
- 故事主要发生在虚构的"海城市"（以深圳、上海为原型）
- 可扩展至：北京、香港、硅谷等商业中心

### 2.2 系统设定

**【未来先知系统】**

主角重生后觉醒的系统，具有以下功能：

| 系统模块 | 功能描述 |
|---------|---------|
| 商业先知 | 每月可预知3件未来商业大事（股票涨跌、公司成败、政策变化） |
| 技能学习 | 消耗"先知点"学习各类技能（商业、格斗、语言、艺术等） |
| 人脉扫描 | 可查看目标人物的未来成就和性格弱点 |
| 魅力光环 | 被动提升异性好感度 |
| 逆天改命 | 可改变小概率事件的结局（需大量先知点） |

**先知点获取方式：**
- 改变重大历史事件走向
- 完成系统任务
- 收集后宫角色
- 建立商业成就

### 2.3 重要地点

| 地点 | 描述 | 剧情关联 |
|------|------|---------|
| 海城三中 | 主角重生后的初中 | 开篇剧情、初恋线 |
| 电子城 | 早期倒货场所 | 第一桶金 |
| 深圳华强北 | 电子产品集散地 | 批发业务线 |
| 海城高新区 | 创业园区 | 公司总部、研发中心 |
| 香港中环 | 金融中心 | 资本运作、IPO |
| 硅谷 | 科技圣地 | 海外扩张线 |

## 3. 角色设计

### 3.1 主角

**林辰（Lin Chen）**

| 属性 | 详情 |
|------|------|
| 年龄 | 开篇13岁（1999年）→ 结局44岁（2030年） |
| 外貌 | 清秀少年，后期成熟稳重，气质不凡 |
| 性格 | 重生前：懦弱、优柔寡断、善良但无主见<br>重生后：果断、精明、腹黑、有仇必报 |

**前世经历：**
- 出生普通家庭，父母是工人
- 学习成绩中等，考上二本大学
- 毕业后进入小公司，一直碌碌无为
- 28岁结婚，妻子是苏婉晴
- 35岁发现妻子与上司出轨，谈判时发生车祸身亡

**重生目标：**
1. 建立商业帝国，成为顶级富豪
2. 让前世背叛的妻子苏婉晴付出代价
3. 保护前世没能保护的人（父母、恩师）
4. 收集前世错过的红颜知己

**系统技能树：**

```
【商业系】
├─ 股神直觉（预知股市走向）
├─ 商业谈判（提升交易成功率）
├─ 管理学识（提升公司运营效率）
└─ 资本运作（并购、IPO相关）

【个人系】
├─ 格斗精通（自保能力）
├─ 语言天才（英/日/法/俄）
├─ 社交魅力（提升人脉获取）
└─ 健康体质（延长寿命）

【特殊系】
├─ 未来科技树（提前获取科技知识）
├─ 预知梦（随机触发未来场景）
└─ 读心术（短暂读取他人想法）
```

## 4. 剧情大纲

### 4.1 序章：死亡与重生

**场景1：车祸前夜**

> 地点：医院门口
> 时间：2019年冬夜

**【CG：雨夜车祸现场】**

林辰站在医院门口，手里拿着离婚协议书。对面是他的妻子苏婉晴和她的情人赵天成。

**苏婉晴**：林辰，你是个好人，但给不了我想要的生活。签了吧，我会给你一笔补偿。

**林辰**：（颤抖）十年了...我为了这个家付出了一切...你就这样对我？

**赵天成**：（嗤笑）别装了，你这种废物也配叫男人？婉晴跟了你十年是浪费青春。

**林辰**：你们...你们会后悔的！

**苏婉晴**：后悔？林辰，认清现实吧。你这辈子也就这样了。

三人在争执中，一辆失控的卡车冲来...

**【黑屏】**
**【音效：刺耳刹车声、撞击声】**

**林辰内心独白**：我不甘心...如果...如果能重来...

---

### 7.3 人际关系系统

**关系等级**：
```
陌生人 → 认识 → 熟人 → 朋友 → 好友 → 挚友
          ↓
陌生人 → 认识 → 暧昧 → 恋人 → 情侣 → 伴侣
          ↓
陌生人 → 认识 → 敌视 → 对手 → 仇敌
```

**好感度影响**：
- 礼物赠送
- 帮助解决困难
- 共同经历
- 对话选择
- 外貌/魅力/财富

---

### 7.4 战斗/竞争系统

**商业对决**：
- 类似回合制战斗
- 双方轮流使用商业手段
- 消耗"行动点"
- 胜负取决于：资金、人脉、策略、运气

**可用的战术**：
| 战术 | 消耗 | 效果 | 反制 |
|------|------|------|------|
| 价格战 | 高 | 打击对方利润 | 提升品质 |
| 挖人才 | 中 | 削弱对方实力 | 股权激励 |
| 舆论战 | 中 | 损害对方声誉 | 公关处理 |
| 法律诉讼 | 高 | 拖延对方 | 和解 |
| 联盟 | 低 | 获得支持 | 分化联盟 |
| 先知 | 特殊 | 看穿对方策略 | 无 |

### 7.5 成长系统

**主角属性**：
```
【基础属性】
├─ 魅力：影响人际关系和好感度
├─ 智力：影响商业决策和学习速度
├─ 体力：影响健康和工作效率
└─ 运气：影响随机事件

【商业属性】
├─ 资金：当前现金
├─ 资产：总资产估值
├─ 声望：社会声誉
└─ 人脉：可调用的人脉资源

【技能属性】
├─ 商业技能
├─ 格斗技能
├─ 社交技能
└─ 特殊技能
```

**升级方式**：
- 完成任务/剧情
- 使用系统先知
- 学习训练
- 特殊事件

---

### 7.6 后宫系统

**后宫管理**：
- 每个女角有独立的好感度和剧情线
- 可查看当前关系状态
- 可选择约会、送礼等互动
- 需要平衡多个女角之间的关系

**好感度阶段**：
```
0-29：陌生人/路人
30-49：认识/普通朋友
50-69：好友/暧昧
70-89：恋人/伴侣
90-100：深爱/生死与共
```

**特殊状态**：
- 黑化：某些角色可能因为被拒绝或背叛而黑化
- 怀孕：成为伴侣后可能怀孕
- 出轨：如果不关心，某些角色可能出轨
- 死亡：特殊情况下角色可能死亡
