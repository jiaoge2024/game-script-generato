# 回声疯人院 - 生存恐怖游戏设计文档

## 1. 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 回声疯人院 (Echoes Asylum) |
| 游戏类型 | 生存恐怖 + 心理恐怖 + 解谜探索 |
| 核心玩法 | 躲避逃脱 + 资源管理 + 精神值系统 + 音频定位 |
| 目标玩家 | 17-35岁恐怖游戏爱好者 |
| 年龄分级 | M级（17+） |
| 游戏平台 | PC / PS5 / Xbox Series X |
| 预计时长 | 10-15小时（初次通关） |

### 核心理念
> "你是一名调查记者，前往传闻中闹鬼的废弃疯人院调查30年前离奇的集体失踪案。当你踏入的那一刻，大门在你身后重重关闭——你发现这里不仅仅是废弃那么简单。黑暗中的东西在窥视你，你的理智在崩溃边缘，而你唯一能依靠的...是声音。"

### 核心玩法特色

**【音频定位系统】**
- 游戏核心机制：玩家必须依靠声音来定位敌人、寻找线索
- 戴耳机体验最佳：立体声音效帮助判断方向和距离
- 声音也是危险：制造噪音会引来敌人

**【精神值系统】**
- 目击恐怖事物、长时间黑暗、受伤都会降低SAN值
- SAN值过低会产生幻觉、听到不存在的声音
- 极低SAN值可能导致游戏角色失控甚至自杀

**【回声机制】**
- 某些事件会在时间线上留下"回声"
- 可以重放过去的场景片段，获取线索
- 但每次回放都会消耗SAN值

## 2. 世界观设定

### 2.1 黑木疯人院（Blackwood Asylum）

**历史背景**：

| 时间 | 事件 |
|------|------|
| 1920年 | 黑木疯人院建成，位于偏僻山区 |
| 1920-1960 | 正常运营期间，收治约2000名患者 |
| 1963年 | 新任院长Dr.维克多上任 |
| 1965-1975 | 传出虐待患者、非法实验传闻 |
| 1978年3月15日 | 集体失踪事件：134人一夜之间消失 |
| 1978年3月16日 | 疯人院被封锁，官方解释为"集体逃离" |
| 1980-2020 | 多次调查记者进入，皆离奇失踪或死亡 |
| 2023年 | 主角接手调查，进入疯人院 |

**地理位置**：
- 位于黑木山脉深处
四面环山，只有一条山路可通行
- 距离最近的小镇50公里
- 手机信号完全无法覆盖

### 2.3 超自然设定

**【怨灵系统】**

疯人院内充满怨灵，分为以下类型：

| 怨灵类型 | 特征 | 危险度 | 应对方式 |
|---------|------|--------|---------|
| 游荡者 | 无目标移动的影子 | ★☆☆☆☆ | 保持安静 |
| 模仿者 | 模仿人类声音诱捕 | ★★★☆☆ | 不理会声音 |
| 撕裂者 | 快速移动，见光即攻击 | ★★★★☆ | 躲藏 |
| 吞噬者 | 无形存在，吸收精神 | ★★★★★ | 光源驱赶 |
| 看守者 | 30年前的疯人院守卫 | ★★★★★ | 无法杀死，只能躲避 |

**【"它们"】**

游戏中的主要威胁，统称为"它们"：
- 原本是人类，被实验改造成了某种东西
- 对光线敏感，但完全黑暗中视力极佳
- 听觉极其敏锐，任何声音都会引来它们
- 无法被常规武器杀死，只能暂时驱赶

## 3. 角色设计

### 3.1 主角

**艾琳·沃克（Erin Walker）**

| 属性 | 详情 |
|------|------|
| 年龄 | 28岁 |
| 职业 | 调查记者 |
| 身高 | 168cm |
| 外貌 | 短发，干练装束，随身携带录音设备和相机 |

**性格特点**：
- 执着：对真相有着偏执的追求
- 勇敢：愿意冒险获取独家新闻
- 独立：习惯独自工作，不依赖他人
- 创伤：童年时曾目睹父亲谋杀母亲，有心理阴影

**装备**：
| 装备 | 用途 | 耐久 |
|------|------|------|
| 手电筒 | 照明 | 电池消耗 |
| 录音笔 | 录制声音，收集线索 | 存储空间 |
| 相机 | 拍摄证据，闪光可驱赶敌人 | 电池消耗 |
| 对讲机 | 接收神秘信号 | 电池消耗 |
| 急救包 | 恢复健康 | 有限数量 |
| 镇定剂 | 恢复SAN值 | 有限数量 |

**能力限制**：
- 没有武器，只能躲避
- 跑动有体力限制
- 受伤后移动速度下降
- SAN值过低会产生幻觉

## 4. 威胁设计

### 4.1 敌人类型

**【它们（The Taken）】**

| 类型 | 外观 | 行为模式 | 应对策略 |
|------|------|---------|---------|
| **徘徊者** | 人形轮廓，皮肤灰白，四肢扭曲 | 缓慢巡逻，听到声音会加速 | 保持安静，绕行 |
| **潜伏者** | 可爬墙和天花板，发出咔哒声 | 从天而降突袭 | 经常抬头看 |
| **狩猎者** | 瘦长鬼影，移动极快 | 锁定目标后会持续追踪 | 躲藏+制造声东击西 |
| **聚合体** | 多个尸体融合而成 | 巨大、缓慢但无法阻挡 | 必须逃避，无法对抗 |
| **院长** | Dr.维克多的化身 | BOSS级，特定场景出现 | 解谜才能逃脱 |

**【特殊敌人】**

#### 收音机人
- 外观：一个带着收音机的轮椅
- 行为：播放随机音乐，吸引玩家靠近
- 真相：是陷阱，靠近后会触发大量敌人

#### 模仿者
- 外观：无
- 行为：模仿主角亲人的声音呼唤
- 目的：诱骗玩家离开安全区域

---

### 4.2 环境威胁

| 威胁类型 | 描述 | 危险度 |
|---------|------|--------|
| **绝对黑暗** | 手电筒没电时的完全黑暗 | ★★★★★ |
| **精神污染区域** | 进入后SAN值快速下降 | ★★★★☆ |
| **迷宫走廊** | 不断变化的走廊结构 | ★★★☆☆ |
| **时间循环区** | 不断重复同一片段 | ★★★★☆ |
| **现实扭曲** | 空间结构异常，鬼打墙 | ★★★★★ |

---

### 4.3 恐怖元素设计

**【视觉恐怖】**
- 跳跃惊吓（Jump Scares）：约20次，分散在关键节点
- 身体恐怖：实验失败品、变异尸体
- 恐谷效应：像人但不是人的东西
- 阴影恐怖：黑暗中移动的影子
- 血腥：手术区的血迹、内脏

**【听觉恐怖】**
- 细微的呼吸声（不知道来自哪里）
- 远处的脚步声（越来越近）
- 儿童的笑声（在不可能有儿童的地方）
- 尖叫声（突然、近处）
- 寂静（最可怕的是绝对的安静）

**【心理恐怖】**
- 孤立感：完全独自一人
- 偏执：无法相信看到的和听到的
- 无助感：敌人无法被杀死
- 疯狂：SAN值下降导致的幻觉
- 负罪感：了解真相后的道德负担

**【彩蛋】**：

## 6. 系统设计

### 6.1 生存系统

**【资源管理】**

| 资源 | 初始值 | 最大值 | 获取方式 | 消耗方式 |
|------|-------|--------|---------|---------|
| 生命值 | 100 | 100 | 急救包 | 受敌人攻击 |
| 精神值 | 100 | 100 | 镇定剂/安全区域 | 恐怖事件/黑暗 |
| 体力 | 100 | 100 | 自动恢复/休息 | 奔跑/躲藏 |
| 手电电池 | 4节 | 10节 | 搜寻 | 照明消耗 |
| 相机电池 | 2节 | 5节 | 搜寻 | 闪光/拍照 |

---

### 6.2 精神值系统（SAN）

**【SAN值下降触发】**

| 触发条件 | SAN下降 | 附加效果 |
|---------|---------|---------|
| 目击敌人 | -5 | 暂时视野模糊 |
| 被敌人追逐 | -10 | 心跳加速（声音） |
| 进入黑暗区域 | -1/秒 | 逐渐产生幻觉 |
| 阅读恐怖文件 | -15 | 手抖（拿不稳东西） |
| 目击尸体/血腥 | -10 | 恶心（画面扭曲） |
| 死亡后复活 | -20 | 幻觉持续更久 |
| 听到低语 | -5 | 耳鸣 |

**【低SAN值效果】**

| SAN范围 | 效果 |
|--------|------|
| 70-100 | 正常 |
| 50-69 | 偶尔听到幻听 |
| 30-49 | 开始出现视觉幻觉 |
| 10-29 | 严重幻觉，敌人变多 |
| 1-9 | 角色开始自言自语，无法奔跑 |
| 0 | 游戏结束（疯狂结局） |

---

### 6.3 音频系统

**【立体声定位】**

游戏的核心机制，玩家必须依靠声音：

**【声音类型】**

| 声音 | 含义 | 应对 |
|------|------|------|
| 脚步声（远） | 敌人在远处 | 继续前进 |
| 脚步声（近） | 敌人接近 | 准备躲藏 |
| 呼吸声 | 敌人在附近 | 保持静止 |
| 尖叫声 | 敌人发现玩家 | 立即逃跑 |
| 机械声 | 可互动物体 | 调查 |
| 水滴声 | 装饰音 | 无需理会 |
| 儿童笑声 | 陷阱 | 不要跟随 |
| 敲击声 | 回声线索 | 调查来源 |

**【玩家制造的声音】**

玩家必须控制自己的噪音：

| 行为 | 噪音等级 | 风险 |
|------|---------|------|
| 行走（慢） | 低 | 安全 |
| 奔跑 | 高 | 会引来敌人 |
| 开门 | 中 | 可能引来附近敌人 |
| 跌倒 | 高 | 引发追逐 |
| 射击/闪光 | 极高 | 引来所有敌人 |

---

### 6.4 隐藏机制

**【回声系统】**

特定区域可以重放过去事件：

**【使用方法】**：
- 找到"回声点"（镜子/水潭/特定物品）
- 交互进入回声模式
- 观看过去片段
- 收集线索

**【代价】**：
- 每次观看消耗SAN值
- 观看时处于无法防御状态
- 某些回声是陷阱

---

**【时间循环】**

隐藏机制，二周目才会显现：

- 某些物品在二周目位置改变
- 对话会暗示循环
- 特定条件下可以打破

---

### 6.5 谜题设计

**【谜题类型】**

| 类型 | 例子 | 难度 |
|------|------|------|
| 音频谜题 | 按照敲击节奏开锁 | ★★★☆☆ |
| 光线谜题 | 用光线照亮特定图案 | ★★☆☆☆ |
| 顺序谜题 | 按正确顺序拉动拉杆 | ★★★☆☆ |
| 隐藏物品 | 用相机拍出不可见之物 | ★★★★☆ |
| 环境谜题 | 利用回声改变过去影响现在 | ★★★★★ |

## 8. 恐怖节奏设计

**【恐怖节奏曲线】**

```
紧张度
 ↑
 │       ╱╲    ╱╲        ╱━━━━━━╲
 │      ╱  ╲  ╱  ╲      ╱          ╲
 │   ╱╲    ╱╲    ╲    ╱              ╲
 │  ╱  ╲  ╱  ╲    ╲  ╱                ╲
 │ ╱    ╲╱    ╲    ╲╱                  ╲
 └──────────────────────────────────────→ 时间
   1    2    3    4    5    6    7    8
   章    章    章    章    章    章    章
```

**【节奏控制】**

| 阶段 | 时长 | 描述 | 目的 |
|------|------|------|------|
| 压抑期 | 10-15分钟 | 低威胁，高氛围 | 建立紧张感 |
| 爬升期 | 5-10分钟 | 威胁逐渐增加 | 提升心流 |
| 爆发期 | 2-5分钟 | 追逐战/高潮 | 释放紧张 |
| 缓解期 | 5-10分钟 | 安全区域 | 让玩家喘息 |
| 重复 | - | 循环以上阶段 | 持续体验 |
