# 游戏脚本生成器 (Game Script Generator)

一个强大的AI驱动的游戏设计文档生成工具，可以一键创建完整的游戏脚本，包括世界观设定、角色设计、剧情大纲、对话脚本和关卡设计。

## 功能特性

- **多种游戏类型支持**：RPG、冒险、视觉小说、恐怖、动作、策略、模拟经营、解谜
- **完整脚本结构**：包含游戏概述、世界观、角色、剧情、对话、关卡等完整模块
- **智能默认值**：即使只输入简单指令，也能生成高质量的游戏脚本
- **静默执行**：无需确认，直接生成，节省时间
- **多种输出格式**：Markdown主文档 + JSON结构化数据（可选）

## 快速开始

### 方式一：斜杠命令

```bash
# 基本用法
/game-script rpg

# 指定主题
/game-script adventure 科幻

# 完整参数
/game-script visual_novel 恐怖 长篇 PC 黑暗风格

# 简写命令
/gsg horror
```

### 方式二：自然语言

```
"生成一个RPG游戏脚本"
"创建一个科幻冒险游戏的剧情设计"
"帮我设计一个赛博朋克风格的游戏世界观和角色"
"写一个关于时间旅行的视觉小说完整脚本"
"设计一个生存恐怖游戏的关卡和剧情"
```

## 支持的游戏类型

| 类型 | 命令关键词 | 推荐主题 |
|------|-----------|---------|
| 角色扮演 | `rpg`, `角色扮演` | 奇幻、科幻、仙侠 |
| 冒险游戏 | `adventure`, `冒险` | 探险、解谜、叙事 |
| 视觉小说 | `visual_novel`, `视觉小说` | 恋爱、悬疑、日常 |
| 恐怖游戏 | `horror`, `恐怖` | 超自然、生存、心理 |
| 动作游戏 | `action`, `动作` | 射击、格斗、竞速 |
| 策略游戏 | `strategy`, `策略` | 战争、文明、经营 |
| 模拟经营 | `simulation`, `模拟` | 城市、农场、人生 |
| 解谜游戏 | `puzzle`, `解谜` | 密室、逻辑、神秘 |

```
└── 7. 系统设计（可选）
    ├── 战斗系统

```

```
output/
├── rpg_fantasy_20240111_143022.md      # 主脚本文件
├── rpg_fantasy_20240111_143022.json    # 结构化数据（可选）
└── ...
```

**文件命名格式：** `[游戏类型]_[主题]_[时间戳].md`

## 配置文件

项目根目录下的 `config.json` 可以自定义默认值：

```json
{
  "defaultGameType": "rpg",
  "defaultTheme": "fantasy",
  "defaultLength": "medium",
  "defaultPlatform": "PC",
  "defaultStyle": "epic",
  "outputDirectory": "output",
  "includeJsonExport": true
}
```

## Python脚本使用（可选）

如果需要更高级的自定义生成，可以使用Python脚本：

```bash
# 基本使用
python scripts/game_script_generator.py --type rpg --theme fantasy

# 指定输出路径
python scripts/game_script_generator.py --type adventure --theme sci-fi --output ./my_game.md

# 批量生成
python scripts/game_script_generator.py --batch --config batch_config.json
```

## 最佳实践

1. **明确需求**：虽然工具可以使用默认值，但提供具体的游戏类型和主题会得到更精准的结果

2. **逐步细化**：先生成整体框架，再使用续写模式逐步完善细节

3. **结合实际**：根据目标平台和开发团队规模，调整脚本复杂度

4. **版本控制**：为每个游戏概念生成独立的脚本文件，便于比较和选择

5. **导出备份**：定期备份 `output/` 目录中的生成文件

## 常见问题

**Q: 生成的脚本可以直接用于游戏开发吗？**
A: 脚本提供的是完整的设计框架和创意基础，具体实现还需要根据游戏引擎和开发计划进行调整。

**Q: 可以生成特定游戏引擎的代码吗？**
A: 当前版本主要生成设计文档。代码生成功能在规划中，未来可能支持Unity、Unreal等引擎的脚本模板。

**Q: 如何修改默认设置？**
A: 编辑项目根目录的 `config.json` 文件，或者直接在命令中指定参数覆盖默认值。

**Q: 支持多语言输出吗？**
A: 支持！可以在请求中指定语言，如"生成一个英文的RPG脚本"或"创建日文视觉小说"。
